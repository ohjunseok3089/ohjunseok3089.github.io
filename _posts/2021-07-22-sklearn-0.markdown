## 사이킷런 이란?

+ 사이킷런 (Scikit-learn)은 현재 실무 및 연구에서 사용되고 있는 파이썬 (Python) 머신러닝 프로그램들 중, 가장 보편적이게 사용되고 있는 라이브러리 입니다. 사이킷런은 머신러닝에 관련된 기능들을 사용자들에게 보다 쉬운 방식으로 접근 및 사용할 수 있게 해주는 라이브러리로, 현재 파이썬에서 머신러닝을 개발 한다는 것은 즉슨 사이킷런을 사용함을 의미할 정도로 널리 쓰이고 있습니다.

  + 저희 또한 이러한 이유로 이 라이브러리를 사용하기로 결정하였습니다. 😊

* * * 

## 사이킷런 설치하기. 
사이킷런을 설치하려면 첫번째로 파이썬을 설치해야 합니다.
그리고, 파이썬에서 데이터 분석 등에 활용되는 라이브러리인 아나콘다 (Anaconda)를 설치해야 합니다.

+ 아나콘다(Anaconda)란?

  + 아나콘다(Anaconda)는 파이썬에서 데이터를 분석할때에 사용할 수 있는 다양한 라이브러리를 포함하는 패키지형 소프트웨어로, 아나콘다 하나를 설치함으로서 데이터 관련 다른 라이브러리들을 별도로 설치해야하는 번거로움이 사라진다.

## 아나콘다(Anaconda) 설치방법

1.  아나콘다 홈페이지에서 설치 패키지 다운로드
    1.  [아나콘다 설치](https://www.anaconda.com/products/individual#macos)
    2.  설치가 완료되었으면, 잘 작동하는지 확인해본다.
    ``` bash
    conda —version
    ```
    3.  또한, 업데이트 진행도 실시해준다.
    ```
    conda update conda
    ```
        만약 conda 명령어가 실행이 되지 않는경우
        ```
        vi ~/.bash_profile
        export PATH=“/opt/anaconda3/bin:$PATH”
        ```
        -   /opt 이후 경로는 본인이 설치한 anaconda3의 경로로 지정해주셔야 합니다.
        -   esc-> :wq 엔터 후, cmd 에서 source ~/.bash_profile 실행

`아나콘다(Anaconda)`를 설치 완료하였으면, 이젠 사이킷런을 설치해야 합니다.

1.  Scikit learn 설치
    ```
    conda install scikit-learn
    ```
2.  Jupyter Notebook 설치
    ```
    pip install scikit-learn
    ```

설치가 완료되면 아나콘다와 동일하게 잘 작동하는지 확인해봐야합니다.

파이썬 확장자의 파일을 하나 만드신 후, 
```
import sklearn
print(sklearn.__version__)
```
을 확인하시면 됩니다.
